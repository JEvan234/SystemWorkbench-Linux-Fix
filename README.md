# SystemWorkbench-Linux-Fix
This repository features a guide to get STM SystemWorkbench running on linux.

# Requirements
- Bottles (Flatpak)
- ST link driver
- STM system workbench **Windows Installer**
- JRE8 MSI file from Adoptium

# Steps

## Installing Pre-reqs

- Be sure to install bottles using flatpak here: [https://flathub.org/en/apps/com.usebottles.bottles](https://flathub.org/en/apps/com.usebottles.bottles)
- Install the System workbench installer (.exe) to a folder you will recall later
- Install the JRE msi to a folder you will remember later

## Installing the Java Runtime Enviroment (JRE)
- Boot up the bottle with the MSI installer
- Follow the instructions
- After it has finished installing, run the .EXE **In the Same Bottle**
