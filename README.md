# SystemWorkbench-Linux-Fix
This repository features a guide to get STM SystemWorkbench running on linux.

# Requirements
- Bottles (Flatpak)
- ST link driver
- STM system workbench **Windows Installer**

# Steps

## Installing Pre-reqs

- Be sure to install bottles using flatpak here: [https://flathub.org/en/apps/com.usebottles.bottles](https://flathub.org/en/apps/com.usebottles.bottles)
- Install the ST link driver
